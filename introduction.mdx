## What is Stremio?

Stremio is a modern media center that serves as a one-stop solution for your video entertainment. You can discover, watch, and organize video content from easy-to-install addons — all in one unified interface.

Stremio Web is the browser-based version of the Stremio client, built with React and powered by the `stremio-core-web` engine. It runs entirely in the browser and can also be self-hosted using Docker.

## Content types

Stremio supports a wide range of media content types through its addon ecosystem:

| Type | Description |
|------|-------------|
| `movie` | Feature films and short films |
| `series` | TV shows with seasons and episodes |
| `channel` | Video channels and shows |
| `tv` | Live TV streams |
| `music` | Music videos and albums |
| `radio` | Live radio stations |
| `podcast` | Podcast episodes and shows |
| `game` | Gaming content |
| `book` | Digital books |
| `adult` | Adult content (age-restricted) |

## Architecture overview

Stremio Web is a single-page application (SPA) built on:

- **React 18** — UI layer with component-based architecture
- **stremio-core-web** — Core business logic, state management, and addon protocol
- **Less** — Styling with CSS modules
- **Webpack** — Build toolchain with code splitting
- **i18next** — Multi-language internationalization
- **Workbox** — Progressive Web App (PWA) service worker

The app communicates with addons through the Stremio addon protocol and optionally connects to a local streaming server for torrent and advanced stream support.
