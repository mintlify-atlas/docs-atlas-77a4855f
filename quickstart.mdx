---
title: "Quickstart"
description: "Get a working Stremio Web instance running in minutes."
---

Stremio Web is a browser-based media center that lets you discover, watch, and organize video content through installable addons. This guide gets you from zero to a running development instance as fast as possible.

<Steps>
  <Step title="Install prerequisites">
    You need **Node.js 20** and **pnpm 10 or higher** before you begin.

    Check your Node.js version:

    ```bash
    node --version
    ```

    The project requires Node.js 20 (as specified in `.nvmrc`). If you use [nvm](https://github.com/nvm-sh/nvm), run:

    ```bash
    nvm install 20
    nvm use 20
    ```

    Then install pnpm if you don't have it:

    ```bash
    npm install -g pnpm
    ```

    Verify pnpm is available:

    ```bash
    pnpm --version
    ```

    <Info>
      pnpm 10 or higher is required. Visit the [pnpm installation docs](https://pnpm.io/installation) for alternative installation methods.
    </Info>
  </Step>

  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/Stremio/stremio-web.git
    cd stremio-web
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    pnpm install
    ```

    This installs all packages defined in `package.json`, including React 18, Webpack, and the Stremio core libraries.
  </Step>

  <Step title="Start the development server">
    ```bash
    pnpm start
    ```

    Webpack starts in development mode and watches your files for changes. Wait until you see a message indicating the build is complete.

    <Tip>
      The dev server includes hot module replacement, so most changes you make to the source are reflected in the browser without a full reload.
    </Tip>
  </Step>

  <Step title="Open the app in your browser">
    Navigate to [http://localhost:8080](http://localhost:8080) in your browser.

    You should see the Stremio Web interface load. If the port is already in use, check your terminal output for the actual URL the dev server is serving on.
  </Step>

  <Step title="Log in or create an account">
    On the Stremio Web landing screen:

    - If you already have a Stremio account, enter your credentials and click **Log In**.
    - If you're new, click **Register** to create a free account.

    <Note>
      A Stremio account syncs your library, settings, and installed addons across devices.
    </Note>
  </Step>

  <Step title="Install your first addon">
    Once logged in:

    1. Click the **puzzle piece icon** in the left sidebar to open the **Addons** screen.
    2. Browse the catalog of official addons or click **Install addon by URL** to add a community addon.
    3. Click **Install** next to any addon to activate it.

    After installing an addon, its content appears in the **Discover** and **Board** views automatically.
  </Step>
</Steps>
